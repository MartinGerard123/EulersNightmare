# Euler's Nightmare 🌙

A Python-based automated factorization engine that targets large integers (RSA) using a tiered algorithmic approach.

### 🚀 Features
* **Tiered Attack Logic:** Automatically selects the most efficient algorithm based on bit-length.
* **Advanced Algorithms:** Includes Pollard's $p-1$, Pollard's Rho (Brent's improvement), and Fermat's method.
* **Pre-processing Suite:** Fast primality testing and perfect power detection to avoid wasted cycles.
* **Session Management:** Built-in timeouts and directory-wide scanning for batch processing.
